Require certain users to change their passwords after a specified length of time.

== Description ==

**Did you find this plugin helpful? Please consider [leaving a 5-star review](https://wordpress.org/support/view/plugin-reviews/expire-passwords).**

* Allows you to set the maximum number of days users can use the same password (90 days by default).
* Allows you to set which user roles will be required to reset their passwords (non-Administrators by default).
* Hardens security by preventing user passwords from going stale.
* Ideal for sites needing to meet certain industry security compliances - such as government, banking or healthcare.

**Development of this plugin is done [on GitHub](https://github.com/fjarrett/expire-passwords). Pull requests welcome. Please see [issues reported](https://github.com/fjarrett/expire-passwords/issues) there before going to the plugin forum.**
